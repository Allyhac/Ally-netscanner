# Ally Network Scanner

A simple Python port scanner built for learning networking and ethical hacking.

## Features

- Scan custom IP
- Custom port ranges
- Verbose mode
- Progress indicator
- Shows open ports

## Usage

Usage:

python scanner.py -i 192.168.1.1
python scanner.py -i 192.168.1.1 -s 20 -e 100
python scanner.py -i 192.168.1.1 -v
python scanner.py -i 192.168.1.1 -f
python scanner.py -i 127.0.0.1 -s 1 -e 1000
